周日 plan

每天一个小乐趣
    成本低，随时能开始。
    例如：
    学一个新的舞蹈动作
    视奏一首新曲子
    尝试一种新的素描材料
    做一道没做过的菜
    去一家新餐馆
    读一本新书的第一章
    体验一个新App
    走一条新路线

    特点：

    30分钟~3小时
    不需要准备
    不求结果
    只求新鲜感

    这是在持续给大脑：

    Novelty（新奇）

每周一个新探索
    例如：

    规划一次国家公园旅行
    学习攀岩
    准备一次演出
    完成一个开源项目PR
    学一首完整的古琴曲
    组织一次朋友聚会
    参加一个比赛

    特点：

    有期待感
    有故事线
    有进展感

    提供的是：

    Anticipation（期待）

    很多幸福感其实来自期待，而不是完成。

每月一个大乐趣
    比如：

    研究奥林匹克国家公园路线
    规划温哥华周末
    规划亲子活动
    规划一次攀岩旅行

    这样周一到周五都会有期待。
每季度一个大冒险
